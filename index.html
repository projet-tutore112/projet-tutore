Zola GUI Project

L'interface graphique qui rend les sites statiques accessibles à tous.

Pourquoi ce projet ?

Les générateurs de sites statiques sont performants mais exigent des compétences techniques. Notre mission : créer un CMS de bureau complet pour gérer Zola sans jamais toucher à la ligne de commande ni au code.

✏️

Édition Riche

Gérez votre contenu, insérez des vidéos ou des carrousels d'images via une interface visuelle intelligente.

⚙️

Formulaires Dynamiques

L'application analyse vos templates et génère les formulaires de saisie automatiquement.

⏳

Machine à remonter le temps

Historique Git intégré. Sauvegardez vos états et annulez vos erreurs en un clic.

🚀

Déploiement 1-clic

Publiez votre site directement sur Internet via FTP ou automatisez-le avec GitHub Pages.
