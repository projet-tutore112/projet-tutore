Architecture du système

Comment marche Zola

L'application est développée avec Electron.js. Elle combine les technologies Web (HTML5/JS) pour l'interface et la puissance de Node.js pour interagir directement avec le système de fichiers de l'ordinateur.

Technologies utilisées

⚛️

Electron

Persistance, Conteneurisation et gestion des utilisateurs.

🌐

HTML5 / JS

API & Logique. Gestion des processus Zola et manipulation des fichiers.

⚡

Zola

Générateur de site statique. Moteur de rendu haute performance.

Fonctionnement Interne

L'interface discute avec le Main Process d'Electron pour effectuer les actions système sécurisées.
